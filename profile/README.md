# Saga Tales

**우리의 사가를, 하나의 이야기씩.**

</div>

---

## 소개

Saga Tales는 두 사람이 운영하는 창업 스튜디오입니다.  
매주 하나의 아이디어를 고르고, 하나의 프로토타입을 만들고, 하나의 도메인을 끝까지 파고듭니다.  
대부분의 Tale은 단편으로 끝납니다. 살아남은 몇 편이 사가가 됩니다.

## 일하는 방식

- **매주 하나의 Tale.** 예외 없이 매주 출시합니다.
- **말이 아닌 증명.** 슬라이드 대신 코드와 사용자 피드백으로 검증합니다.
- **넓이보다 깊이.** 일주일 동안은 그 도메인의 바닥까지 갑니다.
- **미련 없는 종결.** 버려진 Tale이 다음 편의 연료가 됩니다.

## 두 사람

서로 다른 깊이를 가진 두 사람입니다.

한 명은 사용자와 제품의 본질에 진심인 개발자입니다.
한 명은 수학과 AI, 그리고 제1원리에 진심인 기술자입니다.

두 개의 깊이가 만나는 지점에서 하나의 Tale이 시작됩니다.

## Tales

이 조직의 각 레포지토리는 하나의 Tale입니다.
살아남을 것도 있고, 그렇지 못할 것도 있습니다. 모두가 우리에게 무언가를 남깁니다.

우리의 사가는 여기서 시작됩니다.
